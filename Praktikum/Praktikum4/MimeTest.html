MimeTest

Konfigurieren Sie Apache so, dass Dateien vom Typ .ewa mit Excel geöffnet werden. Testen Sie mit untenstehendem Link die korrekte Eintragung des MIME-Typs im Webserver.

Achtung! Rufen Sie diese Datei unbedingt über den Webserver auf (also localhost...). Wenn Sie die Datei per Doppelklick öffnen, wird auch der Link über das Filesystem geöffnet und nicht über Apache mit Mime-Types!

Tragen Sie den MIME-Type "application/vnd.ms-excel ewa" so ein, dass MimeTest.ewa mit Excel geöffnet wird. Falls Firefox die Datei direkt anzeigt, stimmt der MIME-Type nicht. Apache sendet für ihm unbekannte Dateitypen nämlich "text/plain"

Sie müssen nach Änderungen den Cache des Browsers löschen und Apache neu starten!

Prüfen Sie anschließend, was passiert, wenn Sie eine bekannte Dateiendung wie pdf mit dem Mime-Type umdefinieren.
